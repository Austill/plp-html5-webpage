# Accessible & SEO-Friendly HTML5 Web Page

This project is a simple, semantically structured HTML5 web page designed to demonstrate best practices in accessibility and search engine optimization (SEO). It was created as part of a PLP assignment to reinforce foundational web development concepts.

## Features

- **Semantic HTML5 structure**: Uses `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>` for clear content organization.
- **Accessibility**: Includes ARIA roles and attributes to support screen readers and assistive technologies.
- **SEO Best Practices**: Features descriptive headings, meta tags, and human-readable content.
- **No CSS or JavaScript**: Focuses solely on HTML5 for learning purposes.

## Live Demo

View the web page hosted on GitHub Pages:

## Usage

1. Clone or download this repository.
2. Open `index.html` in your web browser to view the page locally.

## Assignment Objectives

- Build a basic web page using only HTML5 semantic tags.
- Apply accessibility enhancements for better navigation and usability.
- Implement beginner-friendly SEO techniques for improved discoverability.
